# Brief

We have to implement helpful feature for users to pin the message in the active chat for a limited time so it stays visible during active discussions. We can add pin button on hovering over message so user can pin the message and also we can give some way to set time period of pinned messages. In the active discussion there are multiple users who are using this functionality so we are defining the way how pinned messages will shown in chat. In the chat top section have pinned message Information and also we can highlight the pinned message for proper separation from other messages. 


# Assumptions

- There can be n number of participants in active discussions
- This feature will available to everyone
- We need to showcase pinned message of any user.
- Pinned Message can be seen by any user.
